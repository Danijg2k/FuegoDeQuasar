Fuego de Quasar

Este proyecto localiza una nave imperial situada en un campo de asteroides. Para ello utilizaremos la información que reciben tres de nuestros satélites rebeldes, especificando los mensajes que cada uno de ellos capta, así como la distancia a la que se encuentran del vehículo enemigo.

Funcionalidades:

  • Consulta - Se triangulará la nave enemiga. Para ello tendremos que introducir la información de nuestros satélites.
  • Registro de consultas - Se mostrarán todas las consultas realizadas, con sus respectivas respuestas.

Tecnologías utilizadas:

  • MySql - DDBB (guardar datos relativos a las consultas realizadas y respuestas obtenidas)
  • Java - Back (desarrollo de la API con los métodos necesarios para tratar con la información)
  • Angular - Front (realización de la página web)
